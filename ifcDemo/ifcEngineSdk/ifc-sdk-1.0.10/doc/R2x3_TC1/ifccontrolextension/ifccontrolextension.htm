IFCCONTROLEXTENSION

The data model consists of the following:

4  Entities
1  Enumerations

The IfcControlExtension schema in the core layer declares basic classes for relating constraints and approvals to any object in the IFC model derived from IfcRoot and instantiating life cycle information or time series schedules related to any object derived from IfcObject.

HISTORY New schema in IFC Release 2.0.
