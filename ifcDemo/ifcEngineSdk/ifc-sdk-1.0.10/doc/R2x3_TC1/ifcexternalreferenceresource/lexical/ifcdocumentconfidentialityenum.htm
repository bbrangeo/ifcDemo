IfcDocumentConfidentialityEnum

Definition from IAI: IfcDocumentConfidentialityEnum enables selection of the level of confidentiality of document information from a list of choices.

HISTORY: New enumeration in IFC 2x

Enumeration

Value Definition
PUBLIC Document is publicly available.
RESTRICTED Document availability is restricted.
CONFIDENTIAL Document is confidential and its contents should not be revealed without permission.
PERSONAL Document is personal to the author.
USERDEFINED
NOTDEFINED

EXPRESS specification:

TYPE IfcDocumentConfidentialityEnum = ENUMERATION OF
( PUBLIC,
RESTRICTED,
CONFIDENTIAL,
PERSONAL,
USERDEFINED,
NOTDEFINED);
END_TYPE;
