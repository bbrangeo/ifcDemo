IfcOrthogonalComplement

Definition from ISO/CD 10303-42:1992: This function returns a direction which is the orthogonal complement of the input direction. The input direction must be a two-dimensional direction and the result is a vector of the same type and perpendicular to the input vector.

NOTE  Corresponding STEP function: orthogonal_complement. Please refer to ISO/IS 10303-42:1994, p.101 for the final definition of the formal standard. 

HISTORY  New function in IFC Release 1.5

EXPRESS specification:

FUNCTION IfcOrthogonalComplement
  (Vec : IfcDirection) 
    : IfcDirection;
LOCAL
  Result : IfcDirection ;
END_LOCAL;
  IF NOT EXISTS (Vec) OR (Vec.Dim <> 2) THEN
    RETURN(?);
  ELSE
    Result := IfcRepresentationItem() || IfcGeometricRepresentationItem () || IfcDirection([-Vec.DirectionRatios[2], Vec.DirectionRatios[1]]);
    RETURN(Result);
  END_IF;
END_FUNCTION;
