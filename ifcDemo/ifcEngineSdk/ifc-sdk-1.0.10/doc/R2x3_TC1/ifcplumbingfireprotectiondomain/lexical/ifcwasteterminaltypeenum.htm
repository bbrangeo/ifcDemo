IfcWasteTerminalTypeEnum

Definition from IAI: The IfcWasteTerminalTypeEnum defines the range of different types of waste terminal that can be specified.

HISTORY: New type in IFC 2x2

Value Definition Pset

WASTEDISPOSALUNIT Electrically operated device that reduces kitchen or other waste into fragments small enough to be flushed into a drainage system. YES
