IfcSoundScaleEnum

Definition from IAI: This enumeration defines the different sound scales:

  • DBA: Decibels in an A-weighted scale
  • DBB: Decibels in an B-weighted scale
  • DBC: Decibels in an C-weighted scale
  • NC: Noise criteria
  • NR: Noise rating
  • USERDEFINED: User-defined sound scale
  • NOTDEFINED: Undefined sound scale
HISTORY: New enumeration in IFC R2.0

EXPRESS specification:

TYPE IfcSoundScaleEnum = ENUMERATION OF
( DBA,
DBB,
DBC,
NC,
NR,
USERDEFINED,
NOTDEFINED);
END_TYPE;
