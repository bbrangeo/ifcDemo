IfcInventoryTypeEnum

Definition from IAI: IfcInventoryTypeEnum defines the types of inventory that can be defined.

HISTORY: New Enumeration in IFC Release 2.0

Enumeration:

Value Definition
ASSETINVENTORY A list of asset instances of type IfcAsset
SPACEINVENTORY A list of space instances of type IfcSpace
FUNITUREINVENTORY A list of furniture instances of type IfcFurniture
USERDEFINED
NOTDEFINED
