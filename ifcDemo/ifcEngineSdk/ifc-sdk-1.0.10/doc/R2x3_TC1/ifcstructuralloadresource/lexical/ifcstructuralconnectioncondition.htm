IfcStructuralConnectionCondition

Definition from IAI: Instances of the entity IfcStructuralConnectionCondition or its respective subclasses shall be used to describe more rarely needed connection properties.

HISTORY: New entity in Release IFC2x edition 2.

EXPRESS specification:

ENTITY IfcStructuralConnectionCondition
ABSTRACT SUPERTYPE OF (ONEOF(IfcSlippageConnectionCondition, IfcFailureConnectionCondition));
Name  :  OPTIONAL IfcLabel;
END_ENTITY;

Attribute definitions:

Name  :  Optionally defines a name for this connection condition.
